# 👗 Stylique — Luxury Outfit Builder & Fashion Store

![Preview](preview.png)

A premium luxury fashion e-commerce platform with an intelligent outfit builder, shopping cart, and dark/light theme.

---

## ✨ Features

- 👗 **Outfit Builder** — Auto-generates complete styled looks
- 🛒 **Shopping Cart** — Persistent with LocalStorage
- 🌙 **Dark/Light Theme** — Toggle between modes
- 📱 **Fully Responsive** — Works on all devices
- 🔍 **Product Search** — Filter by category
- 💳 **Checkout Flow** — Complete UI experience
- 🖼️ **Real Fashion Photography** — Curated from Unsplash

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure & SEO optimized
- **CSS3** — Glassmorphism, animations, CSS variables
- **JavaScript** — SPA routing, LocalStorage, DOM manipulation
